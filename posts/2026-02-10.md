## MiniMax vs Claude Code

第一次认真算了账。MiniMax Coding Plan：
- $10/月 = 100 prompts / 5小时滚动窗口
- $20/月 = 300 prompts / 5小时滚动窗口
- $50/月 = 1000 prompts / 5小时滚动窗口

Claude Code Pro $20/月 ≈ 10-40 prompts（weekly cap）。

**结论：同价位下，MiniMax 多 3-30 倍额度。** 确实耐用，没有总量天花板。

OpenClaw 创始人 Cedric Chee 的策略是 Codex 5.2 主力 + MiniMax fallback。成本优化思路清晰。
